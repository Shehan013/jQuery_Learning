jQuery AJAX Methods

ajax()
To send asynchronous HTTP requests to the server.
load()
To send HTTP requests to load the HTML or text content from the server and add them into the HTML element.
get()
To send HTTP GET requests to load the data from the server.
post()
To send HTTP POST requests to load or submit data to the server.
getScript()
To send HTTP GET request to load the JavaScript file form the server and then execute it.
getJSON()
To send HTTP GET requests to load JSON encoded data from the server.
