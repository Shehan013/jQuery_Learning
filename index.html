JavaScript Libraries Vs JavaScript Frameworks

Library is a collection of pre-written code, that can be used optimized a task.

Examples

  • jQuery - DOM Manipulation
  • React - User Interface and Components
  • D3 (Data-Driven Document) - Data Handling

Framework is a structure that can be used to build software.

Examples

  • Angular/Vue Js/Nuxt Js - Frontend Framework
  • Express - Backend Framework
Adavatages of frameworks
  • Reduces risk of errors
  • Save time
  • More Secure code

When we use librries, we are in charge of the flow, but when we use a framwork, the framework is in charge of the flow. - Inversion of Control - IoC

Introduction to jQuery

jQuery is a light-weight JavaScript library.

Capabilities
  • Manipulate DOM elements, attributes and CSS(styles)
  • Create effects and animations
  • Impelemet AJAX(to load data from the servers without refreshing.)
  • Chanining - Perform multiple actions on an element with a single line of code.
Why jQuery?
  • "Write Less, Do More"
  • Cross browser - Run the same in all browsers.
  • Wide variety of plugins
  • Extentable and easy to use
  • Free
